Colors

Swatches

Most situations would use greyscale (our grey/blue colour swatches, or actual greys), with our orange colour as an accent. When used heavily our brand colour can be overwhelming, so try to reserve it for elements that really need the attention.

The yellow secondary colour is used almost exclusively within UI/web-app environments, where more visual feedback is required.

Presentations

Google Slides Template

Our primary presentation tool is Google Slides and our reference Slides template is linked below.

Although in most cases it will be faster to use an existing presentation as the base for a new one, the reference template remains a good starting-point for layout ideas and will be expanded over time with more slide layouts, icons, graph examples, etc.

Remember

  • Slides offers the function to export to Adobe Acrobat (.PDF) ...and also PowerPoint format (.PPT) if necessary.
  • The template (and presentations based on it) have theme colours pre-set. Try to stick to the theme.
  • Orange should be reserved for highlights.
