Introduction

Google Slides Template

Our primary presentation tool is Google Slides and our reference Slides template is linked below.

Although in most cases it will be faster to use an existing presentation as the base for a new one, the reference template remains a good starting-point for layout ideas and will be expanded over time with more slide layouts, icons, graph examples, etc.

Remember

  • Slides offers the function to export to Adobe Acrobat (.PDF) ...and also PowerPoint format (.PPT) if necessary.
  • The template (and presentations based on it) have theme colours pre-set. Try to stick to the theme.
  • Orange should be reserved for highlights.

PowerPoint

Please avoid using PowerPoint where possible, Google Slides was chosen as it allows for consistent formatting, font-syncing, as well as real-time collaboration on presentations, without needing to email powerpoint files around and copy slides back and forth.

This is a significant workflow optimisation and reverting to PowerPoint negates these benefits.

Copy/Pasting Text

Instead of ‘CMD + V’ for pasting, if you press ‘CMD (or CTRL) + SHIFT + V’ you will paste using the formatting of the target element.

This is useful when pasting text into a correctly formatted target element, from an incorrectly formatted source (including older versions of the template, a website, MS Word, a Google Doc, etc.)

Design: Layout

  • I have added a 'Layout' layout to the template’s master slides
  • Swap to the 'Layout' layout, to display a colourful set of slide padding guides.
    • Blue is the minimum padding for a slide
    • Yellow is the target padding for a slide
    • Pink is for slides with less content, or for a hierarchical indentation
  • When building a new slide, it is always a good idea to start with the ‘Layout’ layout, and then switch to the ‘Default’ layout when you’re finished.
  • The ‘Layout’ slide in the template contains the shape elements which can be copy/pasted temporarily, for quickly figuring out padding.

Design: Whitespace

Whitespace is very valuable and does not exist only to be filled by content. Whitespace offers tiny mental breaks from an overload of information, and allows the necessary distinction between groups of content.

Feel free to approach the Design team directly and we will be happy to explain in more detail the purpose of whitespace in a layout.

  • Keep to the template’s design cues regarding spacing from the edge of the slides.
  • The template includes image placeholders in the design. Please resist the urge to add background images everywhere.

Background Images

Charts, infographics, flowcharts and articles are very information-dense and should never have a background image. This adds unnecessarily and detrimentally to the visual information the reader must process within that slide.

Design: Density

Humans are limited in the amount of information they can process at once.

  • Less is more. Question the value of every addition to your presentation.
  • Where possible, try and break up topics into smaller chunks which can be spread across multiple, less-dense slides.
  • Be concise, avoid having any more text than is necessary to convey the message.

Alignment

  • Works on groups as well as single elements.
  • Select multiple elements → Right-click:
    • ‘Align horizontally/vertically’ will align all elements relative to each other.
      Eg. align horizontally left, all elements in the selection will have their left edges aligned, and positioned on the left-most element (vertical positions remain the same).
    • ‘Distribute horizontally/vertically’ will use the most extremely positioned elements as reference for the distribution of the others.
      Eg. distribute horizontally, the left and right-most elements will be used as reference, and the other elements (or groups) will be evenly spaced between them.
  • In the case of a single element (or group) selection, there is also the possibility to ‘Centre on Page’. This does what it says.


Snapping

  • For consistent positioning of elements on the slide, the snapping mode can be swapped between Grid and Guides.
  • Guides is the default mode.
    This mode tries to align and resize elements based on other elements on the slide.
    • Use Guides mode for resizing icons to all have the same height as each other, or for aligning an icon to the centre point of a neighbouring text-box.
  • Grid snaps to the invisible grid that the template is based on.
    This is best for positioning elements consistently and resizing text-boxes, without relying on Slides to correctly guess what you are trying to line it up with.
    • Use Grid mode when cropping and resizing images, positioning elements generally on the slide, resizing shapes/boxes, etc.

Images

Resize

  • Hold ‘SHIFT’ as you drag the handles of an image, to maintain the correct ratio between width and height

Crop

  • Double-click on an image to enter cropping mode
  • Once in cropping mode, the image can be resized, repositioned and rotated separately to the crop.
  • You can crop to a shape by clicking the arrow next to the crop icon on the toolbar and selecting the shape you want.

Replace Placeholders

  • Right-click on an image placeholder and select ‘Replace Image’

Transparency

  • Select an image, then click the ‘Image Options’ button on the toolbar. This will open up a panel on the right where you can change the image opacity.

Groups

Grouping elements allows them to be repositioned and centred based on the dimensions of the group. It can also make duplicating a set of related elements easier.

Group

  • Select multiple items → right-click one of them → select ‘Group’

Ungroup

  • Select multiple items → right-click one of them → select ‘Ungroup’

Edit grouped elements, icons

  • The easiest way to edit icons or grouped elements is to ungroup the group, edit/replace the element, and then regroup it.
  • Alternatively, after selecting a group, clicking again on an element will select that element within the group, allowing you to make changes to the element without affecting the other elements of the group.
